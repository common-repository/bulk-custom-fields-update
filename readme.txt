Change your post custom fields by bulk update.

== Description ==

You can update one field for multiple posts across all post types or you can select multiple custom fields to be updated at once.

== Installation ==

1. Upload `bulk-custom-fields-update` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

`Can i create new custom fields with thi plugin?`

No, plugin works just with existing fields.
